# Brand Guidelines — Vérone Group

Documentation narrative ("le pourquoi") de chaque marque interne du groupe.

⚠️ Marques avec orthographe sensible :

- **Boêmia** (avec accent circonflexe sur le ê) — pas Bohemia, pas Bohême
- Slug technique : `boemia` (sans accent, pour systèmes de fichiers)

## Fichiers attendus

- `brand-verone.md` — Vérone Shop (e-commerce direct, premium éditorial)
- `brand-boemia.md` — Boêmia Shop (bohème + plage méditerranéenne)
- `brand-solar.md` — Solar Shop (luminaires + tech sleek)
- `brand-flos.md` — Flos Shop (plantes séchées + bougies + hygge)
- `brand-linkme.md` — LinkMe (plateforme d'affiliation grand public)
- `brand-office.md` — Office (back-office B2B admin)

## Format attendu (par marque)

- Brand archetype
- Target persona
- Voice & tone
- Mood board verbal (5-10 mots)
- Narrative tagline
