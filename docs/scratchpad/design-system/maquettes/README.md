# Maquettes design system — référence visuelle

Code HTML/JSX des maquettes générées sur claude.ai web (PHASE C du design system).
Sert de référence visuelle pour l'implémentation finale dans les apps.

## Structure

- `verone/` — maquettes Vérone Shop
- `boemia/` — maquettes Boêmia Shop
- `solar/` — maquettes Solar Shop
- `flos/` — maquettes Flos Shop
- `linkme/` — maquettes LinkMe
- `office/` — maquettes Back-office

- Hero / landing

- Card produit (3 variants)
- Page produit
- Footer
- Boutons (primary/secondary/ghost)
- Inputs
