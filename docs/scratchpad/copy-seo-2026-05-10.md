# Copy SEO — Title tags + Meta descriptions + Schema

**Date :** 2026-05-10  
**App :** `apps/site-internet`  
**À lire avec :** `seo-strategy-2026-05-10.md` (keyword clusters + architecture)  
**Usage :** Intégrer dans `generateMetadata()` de chaque `page.tsx`

Règles strictes :

- Title : 50–60 caractères · mot-clé racine EN DÉBUT
- Meta description : 140–155 caractères · mot-clé secondaire inclus · CTA implicite
- H1 : mot-clé racine présent, naturellement intégré (pas de bourrage)
- Schema.org : obligatoire sur chaque type de page

## Fiche produit — `/produit/[slug]` (dynamique)

```
TITLE (pattern dynamique, viser 50-60 car.)
[Nom produit court] — [Matière/Style si court] | Vérone
ex : "Vase céramique nervuré Brun | Vérone"
ex : "Lampe de table pied laiton verre | Vérone"

META DESCRIPTION (pattern dynamique, 140-155 car.)
→ Prendre les 2 premières phrases de la description produit en DB
→ Tronquer à 155 caractères, ne pas couper en milieu de mot
→ Si description vide : "[Nom produit] — sélectionné pour sa matière, sa
   proportion, sa façon de tenir l'espace. Livraison France."

H1
[Nom produit complet exact tel qu'en DB]

KEYWORD RACINE : [catégorie produit] + [matière] + "design"
ex fiche vase : "vase décoratif céramique"
ex fiche lampe : "lampe de salon design"
```

## Schema.org — par type de page

### Homepage + À-propos

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Vérone",
  "url": "https://veronecollections.fr",
  "description": "Sélection éditoriale de déco et mobilier design.",
  "logo": "https://veronecollections.fr/logo.png"
}
```

### Fiche produit

```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "[Nom produit]",
  "description": "[Description produit]",
  "image": "[URL image principale]",
  "brand": { "@type": "Brand", "name": "[Marque produit]" },
  "offers": {
    "@type": "Offer",
    "priceCurrency": "EUR",
    "price": "[Prix TTC]",
    "availability": "https://schema.org/InStock",
    "seller": { "@type": "Organization", "name": "Vérone" }
  }
}
```

### Catalogue + Collections

```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Accueil",
      "item": "https://veronecollections.fr"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Catalogue",
      "item": "https://veronecollections.fr/catalogue"
    }
  ]
}
```
