## Introduction

Workflows mouvements de stock dans le système Vérone : entrées, sorties, transferts, inventaires.

**À documenter** :
- **Entrées** : Réception fournisseur, returns clients
- **Sorties** : Expédition commandes, casse/perte
- **Transferts** : Entre emplacements/magasins
- **Inventaires** : Comptage physique, ajustements
- Triggers automatiques mise à jour `stock_quantity`
- Logs mouvements (`stock_movements` table)
- Permissions Owner vs Admin
